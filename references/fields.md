# 字段 ID 映射参考

## 如何获取你的字段 ID

AI 表格中每个字段都有一个唯一的 ID。打开 AI 表格设计模式（Design Mode），点击字段，在属性面板中可以看到字段 ID。

字段 ID 通常是一串字母和数字的组合，如 `01ZM8y7`、`pMvAy3f` 等。

## 字段 ID 说明

| 字段用途 | 类型 | 说明 |
|---------|------|------|
| 任务名称 | primaryDoc | 任务标题（表格主键），如"本周主要工作任务" |
| 任务情况更新 | text | 每周更新的详细进展内容 |
| 任务状态 | singleSelect | 选项：已完成 / 刚启动 / 未完成 |
| 负责人 | user(multiple) | 可多人，写入时用 `[{"userId":"...","corpId":"..."}]` |
| 所属部门 | singleSelect | 任务归属部门 |

## 任务状态选项

| 名称 | 说明 |
|------|------|
| 已完成 | 任务已完成 |
| 刚启动 | 任务刚开始推进 |
| 未完成 | 任务尚未完成 |
